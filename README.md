# azure-adf-metadata-driven-ingestion
This project demonstrates an enterprise-style metadata-driven ingestion framework built using Azure Data Factory (ADF), Azure SQL Database, and ADLS Gen2.  Instead of creating separate pipelines for each source table, a single generic pipeline dynamically processes multiple tables using metadata configurations.
